# NauatApp
Application for displaying the Nauat restaurant menu.
The main purpose of displaying the menu is offline.
The functionality of product search and order basket has been introduced.

# Stack
- MVVM
- Room
- Firestore
- DI (Koin)
- Coordinator Layout
